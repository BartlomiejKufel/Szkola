#komendy w bash
npm i [nazwa frameworka] - instalacja frameworka np. ejs, mongoose, body-parser lub expressa
npm init -y - stworzenie pliku package.json
cd (./, ../) - poruszanie się po scieżkach
npm i nodemon --save-dev  - instalacja pakietu nodemon
npm start - komęda do uruchomienia aplikacji przez nodemona po konfiguracji package.json

#react
npx create-react-app . stworzenie szablonu projektu w react
